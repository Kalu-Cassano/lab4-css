Over 10,000 downloads of ‘Super Thief’ game in 2016

Unlike traditional claw machine games that operate offline or at game centers, the "Super Thief" game is the first in Vietnam to allow users to interact with a physical claw machine device using the Internet. When successful, players receive rewards like phone cards.
